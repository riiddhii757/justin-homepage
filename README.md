# Just In – Homepage Reimagination

This project is my implementation of the **Frontend Engineer Intern design task** for Just In.

The goal of this assignment was to recreate and reimagine the homepage UI based on the provided Figma mockups while focusing on 
responsive design, interactive scrolling, and modern UI practices.

- Responsive homepage layout

## Getting Started

To run this project locally:

```bash
git clone https://github.com/riiddhii757/justin-homepage.git
cd justin-homepage
npm install
npm run dev
```
The development server will run at:
```
http://localhost:5173
```
